The various iterators this crate provides.

These iterators are not a very stable interface and you really should avoid considering them to be concrete types. A lot of the iterators in this crate use impl Iterator for this reason but restrictions in the language don’t allow this to be used in all places on the versions of rust this crate wants to compile for.
